## Project Overview
An interactive full-stack conversational application built to simulate the complex reasoning, rhetoric, and perspectives of the philosopher Immanuel Kant.

## Tech Stack
* **Language:** Python
* **Frameworks:** FastAPI/Flask (Backend), Basic HTML/JS (Frontend)
* **Cloud Infrastructure:** Azure Cloud Services

## Key Features & Solutions
* Programmed the core NLP chat logic using **Python** to mirror deterministic philosophical responses.
* Engineered both the modular backend logic and a clean, responsive frontend user interface.
* Containerized and deployed the full application infrastructure live on the **Azure cloud network**.
