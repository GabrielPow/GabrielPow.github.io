## Project Overview
Built an autonomous, production-ready data pipeline engineered to automate the heavy lifting of data extraction, cleansing, and formatting.

## Tech Stack
* **Language:** Python
* **AI & Orchestration:** Gemini API
* **Data Formats:** CSV, JSON

## Key Features & Solutions
* Designed an independent data pipeline following a decoupled **multi-agent architecture**.
* Integrated the **Gemini API** to orchestrate intelligent text parsing and structural categorization.
* Automated the ingestion of raw, unstructured data and processed it seamlessly into clean, production-ready **CSV registries**.

## Quantifiable Results
* Eliminated manual data entry by building a 100% automated extraction and filtration framework.
