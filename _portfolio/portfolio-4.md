## Project Overview
Developed as part of an agile development squad, this project delivers a highly functional administrative platform tailored to maximize business value for small teams.

## Tech Stack
* **Frontend:** JavaScript (Vite)
* **Backend:** Python, REST APIs, OpenAI
* **Cloud Infrastructure:** Azure 

## Key Features & Solutions
* Collaborated inside an **agile environment** to architect an end-to-end admin web portal.
* Integrated interactive business intelligence dashboards to monitor team performance and track operational metrics.
* Built a dedicated **AI Agent layer** designed to analyze dashboard data and automatically output tactical business action plans.
